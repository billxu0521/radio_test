# 台灣無線電題庫記憶卡片

業餘無線電人員資格測試練習工具，使用閃卡方式幫助記憶題庫。

## 功能

- 支援一等、二等、三等無線電執照題庫
- 六大分類：法規、系統原理、通訊方法、干擾預防、電磁相容、安全防護
- 閃卡翻轉顯示答案
- 鍵盤快捷鍵：← → 切換題目，空白鍵翻轉
- 題號跳轉功能
- 響應式設計，支援手機與桌面

## 題庫統計

| 等級 | 題數 |
|------|------|
| 一等 | 868 題 |
| 二等 | 528 題 |
| 三等 | 514 題 |

## 技術架構

- Vue 3 + TypeScript
- Vite
- TailwindCSS
- Pinia

## 開發

```bash
# 安裝依賴
npm install

# 啟動開發伺服器
npm run dev

# 建置
npm run build
```

## 授權

### 題庫

本試題庫為公有領域。試題係國家通訊傳播委員會依電信管理法以及業餘無線電人員及電臺管理辦法等法令所舉行之「業餘無線電人員資格測試」使用。依據中華民國著作權法第 9 條第 1 項第 5 款規定，「依法令舉行之各類考試試題及其備用試題」不得為著作權之標的。
